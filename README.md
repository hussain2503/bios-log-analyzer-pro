# BIOS Log Analyzer Pro

## 📌 Description
A Python-based tool to analyze BIOS debug logs and detect errors, warnings, and critical issues with automated reporting.

## 🚀 Features
- Regex-based log parsing
- Severity scoring
- Boot issue detection
- HTML report generation
- CLI support

## 🛠 Tech Stack
- Python

## ▶️ How to Run
```bash
python src/main.py --file logs/bios_log.txt
